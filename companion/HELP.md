## Pixelhue F Series

This module will allow you to control the following Pixelhue F Series products: F4, F4 Lite, and F8.

**Supported Device Versions:**

- F4: V2.4.2.0.STD
- F8: V2.4.2.0.STD
- F4 Lite: V3.2.4.0.STD

### Configuration

- Enter the IP address of the device in the configuration settings, and select the model of the device (F4, F4 Lite, or F8).

**Available actions:**

- Take
- Cut
- FTB
- Freeze
- Preset
- Preset Destination (PVW - PGM)

**Available Feedbacks:**

- FTB Status Detection
- Freeze Status Detection
- PGM Status Detection
